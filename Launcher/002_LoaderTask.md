# LoaderTask

`LoaderTask`类注释如下：

```xml
/**
 * Runnable for the thread that loads the contents of the launcher:
 *   - workspace icons
 *   - widgets
 *   - all apps icons
 *   - deep shortcuts within apps
 */
```

表示其用来加载Launcher中的内容：

+ 桌面icon
+ 小部件
+ 所有app icon
+ deep shortcuts within apps
