# Launcher3

## 更新：20220821
* Android-13 (Launcher-master-20220821(b62600b8c3de47a26386aa59f06959b2cf1af753)版本源码)

## 说明
目前只支持withoutQuickstep编译
如果因为Android studio或者Intellij Idea版本问题可以使用Intellij Idea 2022.1.2版本可以正常编译。
