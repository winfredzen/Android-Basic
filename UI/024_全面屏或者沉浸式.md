## ActionBar

### 隐藏ActionBar

1.使用style，继承自

```xml
Theme.AppCompat.NoActionBar（暗色主题）
```

或者

```xml
Theme.AppCompat.Light.NoActionBar（亮色主题）
```

2.代码方式

`Activity` 继承自 `AppCompatActivity` 时

```java
getSupportActionBar().hide();//这种方式默认式亮色主题
```

`Activity` 继承自 `Activity` 时

```java
getActionBar().hide();
```
